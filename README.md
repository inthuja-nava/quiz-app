## WEEK 5 ASSIGNMENT - QUIZ APP 

This assignment is an interactive quiz application built using HTML, CSS, and JavaScript. The quiz presents with multiple-choice questions, provides immediate feedback after each answer, tracks the user's score, and displays final results screen with a percentage and letter grade. 

The quiz topic chosen for this project is Space & Astronomy.
